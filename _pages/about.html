About The Frederick

{% endregionblock %} {% regionblock history_heading_2, type:header %}
A Historic Tribeca Hotel
{% endregionblock %} {% regionblock history_content, type:text %}

One of the longest-running historic hotels in Tribeca, The Frederick was built in 1838 in the Gothic Revival style, with some Italianate details added soon after. With many of these details still intact, The Frederick is able to express its history while still leaving room for stylish reinvention.
